# 别花乐 v0.4.0 发布说明

**Mist Teal Ledger** 视觉全量重构：安静的私人账本，看得清、记得快。

## 你能注意到的变化

- **账单首页**：一眼看到「本月已花」，月份与大数字居中；分类/趋势/最近流水往下滚
- **记账页**：支出/收入/转账指示条 + 居中金额舞台，类别选中更轻
- **设置**：分组列表，去掉卡片堆叠；可开关「跟随壁纸取色」（默认关，保持品牌青绿）
- **字体与氛围**：标题偏宋、金额等宽、雾青纸感底（深色为墨绿炭）

## 未改动的部分

- 记账规则、备份导入、软删/回收站、Room schema（仍为 v3）
- 不上云、不联网

## 安装

安装新 APK 即可；主题偏好会保留，新增 Dynamic Color 项默认为关。
