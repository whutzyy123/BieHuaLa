## 0. 为什么需要这份文档

当前实现以 Material 3 默认组件 + 系统字体 + 扁平 `#FCFCFC` 底 + 卡片堆叠为主，功能可用，但观感接近「脚手架 Demo」：

| 现状问题 | 对用户的感受 |
|----------|--------------|
| 系统默认字体 + Monospace 金额 | 无品牌记忆，金额区像调试界面 |
| 背景近纯白 / 纯黑 | 冷、空、像未完成皮肤 |
| SectionCard / 多 Card 堆叠 | 设置与账单像后台控制台 |
| Dynamic Color 优先 | 品牌松石绿被壁纸色冲掉 |
| 信息密度高、分区弱 | 「一眼看账」被工具栏噪音盖住 |

本文件定义**可执行的视觉与构图规范**，后续改 UI 以本文为准；与 PRD 冲突时，**产品规则以 PRD 为准，视觉表达以本文细化 PRD §7.1**。

## 1. 产品调性（设计从这里长出来）

### 1.1 一句话

**安静的私人账本**：记一笔不费劲，看一眼就安心——不上云、不打扰、不炫技。

### 1.2 气质关键词

| 要 | 不要 |
|----|------|
| 松弛、清晰、克制 | 金融仪表盘、加密货币霓虹 |
| 干净、有呼吸感 | 信息墙、统计贴纸满屏 |
| 温和积极（「乐」） | 恐吓式红色警报铺满 |
| 本地手账质感 | SaaS 营销落地页 |

### 1.3 品牌信号测试

任意主屏（尤其账单首页）在去掉底部导航后，仍应能通过：

1. **松石绿主色氛围**  
2. **「别花乐」或等价品牌字样（设置/关于以外的关键时刻可弱出现）**  
3. **金额数字的专属字体与层级**

判断为「换个名字也能是别的记账 App」→ 品牌过弱，需加强。

## 2. 视觉方向（选定方案）

**方向名：Mist Teal Ledger（雾青账本）**

- **主色**：保持 PRD 松石绿 `#006C5C`，强调 `#6FF7DD` 仅用于焦点与少量高光，不作大面积填充。  
- **纸感底**：浅色模式用极淡青灰渐变 + 微噪点（不是纯白）；深色模式用墨绿炭底，避免纯 `#121212` 塑料感。  
- **结构**：默认**少卡片**；列表行与分区用间距 / 细分割线建立层级，卡片仅用于「必须成块点击」的交互容器（如确认区、筛选底栏）。  
- **Material 3**：继续用 Compose M3 组件能力，但 **ColorScheme / Typography / Shape 全面品牌化**，禁止裸奔默认主题。  
- **Dynamic Color**：**默认关闭品牌优先**；设置项可保留「跟随壁纸取色」为可选（默认关），避免品牌漂移。

### 2.1 明确避开的 AI 套路脸

- 紫白渐变 / 紫到靛蓝  
- 暖奶油底 + 衬线大标题 + 陶土强调色  
- 报纸细线多栏密排  
- 大面积霓虹描边、多层阴影、胶囊标签墙、emoji 当图标

## 3. 设计 Token

### 3.1 色彩

#### 品牌（fallback，Android 11- 与「品牌优先」模式）

| Token | Light | Dark | 用途 |
|-------|-------|------|------|
| `brand.primary` | `#006C5C` | `#6FF7DD` | 主按钮、选中 Tab、关键控件 |
| `brand.onPrimary` | `#FFFFFF` | `#005047` | 主色上的字 |
| `brand.container` | `#D0F5EE`（柔化后的青容器，替代刺眼 `#6FF7DD` 铺底） | `#005047` | 轻强调底 |
| `brand.ink` | `#0E1F1C` | `#E6F2EF` | 主文字偏冷青黑 |

#### 语义（账本必备，不随 Dynamic Color 漂移）

| Token | Light | Dark | 用途 |
|-------|-------|------|------|
| `semantic.expense` | `#B42318` | `#F97066` | 支出金额、支出模式 |
| `semantic.income` | `#027A48` | `#6CE9A6` | 收入金额、收入模式 |
| `semantic.transfer` | `#175CD3` | `#84CAFF` | 转账金额、转账模式 |

> 语义色只服务「钱的方向」，不要拿去染整页背景。

#### 表面与氛围

| Token | Light | Dark |
|-------|-------|------|
| `surface.base` | `#F3F7F5` → `#EAF3F0` 竖向微渐变 | `#0C1412` → `#121A18` |
| `surface.elevated` | `#FFFFFF` @ 92% 不透明度叠在 base 上 | `#1A2421` |
| `surface.muted` | `#DDE8E4` | `#24302C` |
| `stroke.subtle` | `#C5D5D0` | `#2F3D39` |
| `text.secondary` | `#5B6B66` | `#A3B4AF` |

可选：`surface.base` 上叠加 **2–3% 噪点**（Compose 用极淡 `Brush` 或预烘焙 drawable），形成「纸」而不是「塑料白」。

### 3.2 字体

| 角色 | 建议 | 说明 |
|------|------|------|
| **品牌 / 大标题** | `Noto Serif SC`（或思源宋体）Medium/SemiBold | 「别花乐」与月份标题；有温度、非系统默认 |
| **正文 / UI** | `Noto Sans SC` | 清晰易读；替代 `FontFamily.Default` |
| **金额** | `IBM Plex Mono` 或 `JetBrains Mono`，**tabular figures** | 列表与大数字对齐不跳动 |

层级建议（sp，可按设备微调）：

| 样式 | 字号 | 字重 | 用途 |
|------|------|------|------|
| `money.hero` | 40–48 | SemiBold | 账单「本月已花」 |
| `money.row` | 17–18 | Medium | 流水行金额 |
| `title.screen` | 22 | Serif SemiBold | 页标题 / 月份 |
| `body` | 15–16 | Regular | 说明、列表主文 |
| `caption` | 12–13 | Regular | 次要元数据 |

**禁止**：整 App 只用系统默认 + 调试感 Monospace。

### 3.3 间距与圆角

| Token | 值 | 用途 |
|-------|-----|------|
| `space.xs` | 4 | 紧贴元素 |
| `space.sm` | 8 | 行内 |
| `space.md` | 16 | 屏边距、模块间距基线 |
| `space.lg` | 24 | 大区块 |
| `space.xl` | 32 | 首屏呼吸 |
| `radius.sm` | 10 | 输入框、小控件 |
| `radius.md` | 16 | 底栏 sheet、必要容器 |
| `radius.lg` | 24 | 极少使用的大面板 |

底部 NavigationBar：顶部分割用 `stroke.subtle`，**不要**厚重 elevation 阴影堆叠。

### 3.4 图标

- 使用 **简洁线性 / 双色** Material Symbols（或等价），描边约 1.5–2dp。  
- 类别仍可用预设色点 + 短标签；**不要用 emoji 当面图标**（列表可读性与风格一致性差）。  
- 账户/类别色点：直径 8–10dp 圆，描边可选 1dp `stroke.subtle`。

## 4. 动效原则（有存在感，不吵）

每个主流程至少保留 **2–3 个有意动效**，统一时长与曲线：

| 动效 | 规格 | 出现位置 |
|------|------|----------|
| **金额入场** | 120–180ms，fade + 轻微上移 4dp | 账单 hero 数字、保存成功后 |
| **模式切换** | Tab 指示条 200ms `FastOutSlowIn`；金额色 crossfade | 记账 支出/收入/转账 |
| **列表变更** | 软删后 item 高度收起 180ms | 账单 / 全部流水 |
| **Sheet** | 标准 M3 容器变换即可，避免弹跳 overshoot | 筛选 |

禁止：循环闪烁、强弹跳、全屏粒子、保存时整页闪白。

---

## 5. 导航与信息架构（视觉层）

保持 PRD 三 Tab：**账单 | 记账 | 设置**。视觉约束：

1. **底部 Tab 永远可见**（已实现）——选中态用品牌色图标 + 短标签，未选中降对比，不要「胶囊底座」堆满。  
2. 子路由（详情、全部流水、账户管理）时，父 Tab 仍高亮（已实现），顶栏用简洁返回 + 标题。  
3. **记账 Tab 是高频动作页**：首屏应让「金额 + 类别 + 保存」成为绝对焦点，次要字段下沉。

## 6. 分屏构图规范

### 6.1 账单 Tab（首页）—— 一个主任务：看清这个月花了多少

**首屏（第一视口）只保留：**

1. 月份切换（轻：左右 chevron + 居中「2026年7月」Serif）  
2. **Hero 支出金额**（`money.hero` + 语义支出色或主文字，按产品偏好：大数字用墨色、符号用支出色）  
3. 一行辅助：收入 · 结余（次级字色，勿再套大卡片描边）  
4. 可选：一条极细支出/收入比例轨（高度 4dp，圆角满宽，不要进度条特效）

**首屏不要出现：** 饼图、折线、筛选芯片墙、多块统计卡、地址/促销类信息。

**向下滚动再出现（每块一件事）：**

| 区块 | 标题 | 内容原则 |
|------|------|----------|
| 分类 | 「花在哪」 | 饼图缩小视觉权重；点击扇区/图例 → 全部流水并带上该 `categoryId`；空态一句话 |
| 趋势 | 「怎么花的」 | 单折线；粒度切换用文字 Segment，不用三枚大 Chip |
| 最近 | 「最近」 | 最多 5 行；行高宽松；金额右对齐 tabular；**无**搜索/筛选（查工具在全部流水） |
| 出口 | 「本月流水」弱链（Hero 下）+「查看全部」 | 同进全部流水；非第二套 FAB |

列表行结构：`色点 | 类别名 + 说明` 左；`金额 + 日期` 右。**不要**每行一张 Card。单击进详情；长按 → 编辑 / 删除。

### 6.2 记账 Tab—— 一个主任务：30 秒内记完

**竖向节奏（上 → 下）：**

1. 模式三段：**支出 | 收入 | 转账**（下划指示 + 语义色，不是三块填色按钮）；编辑态锁定模式  
2. **金额舞台**：大数字居中偏上，背景可有极淡品牌光晕（不是霓虹）  
3. 类别网格：4 列，选中用 `brand.container` 底 + 主色描边；未选中无重边框  
4. 账户 / 时间 / 说明：单行字段，统一 `radius.sm`  
5. **保存**：宽主按钮贴键盘上方；新建保存后留在本页连记；编辑保存后 pop  

转账模式：双账户同一行「出 → 入」，金额色用 `semantic.transfer`。

### 6.3 全部流水 / 详情

- **查/搜/筛**集中在此页：顶栏搜索 + 筛选；筛选勾选即时生效，Sheet 底部「完成」关闭；已筛选态用图标旁品牌色小圆点。  
- 详情：上半金额 hero，下半键值列表；主 CTA「编辑」用实心按钮，删除保持危险色文字降权。

### 6.4 设置

**取消「大卡片套小卡片」的后台感。** 改为：

- 分组标题（caption + 字距）  
- 行列表：左标题、右说明/chevron，分组之间 `space.lg`  
- 关于区可出现品牌词「别花乐」Serif 一次，副文案「完全本地 · 不联网」

备份 / 主题 / 回收站：保持现有信息架构，只换皮与间距。

### 6.5 账户 / 类别管理

- 列表行：色点 + 名称 + 余额（账户）；滑动或菜单归档  
- 编辑对话框：字段纵向；颜色/图标选择复用 `IconColorPickerSection`，样式对齐新 token（色点更大、选中描边用 `brand.ink`）

## 7. 组件级约定（Compose）

| 组件 | 约定 |
|------|------|
| 主按钮 | `primary` 实心，高度 48–52，`radius.md`，无多层阴影 |
| 次按钮 | TextButton / Outlined，描边 `stroke.subtle` |
| 输入框 | 填充 `surface.elevated`，聚焦描边 `brand.primary` |
| Snackbar | 深墨底 + 浅字，或品牌容器；短句 |
| 空态 | 居中插画可用简线图标 + 一句标题 + 一句引导；少按钮 |
| 图表 | 线宽 2dp；饼图最多 5+其他；颜色来自类别 hex，缺省用品牌序列 |

新增文件建议（落地阶段，非本文必含代码）：

- `ui/theme/Color.kt` — 扩展 token  
- `ui/theme/Type.kt` — 接入字体资源  
- `ui/theme/Shape.kt` — 圆角  
- `ui/theme/Background.kt` — 氛围渐变  
- `res/font/` — Noto / Plex 等

## 8. 浅色 / 深色

| | 浅色（默认气质） | 深色 |
|--|------------------|------|
| 情绪 | 晨雾青、纸质账本 | 夜读、护眼墨绿 |
| 金额 | 高对比墨色 | 高对比浅青白 |
| 图表 | 略降饱和 | 略提明度，避免霓虹 |

手动主题切换保留；**品牌优先模式下 Light/Dark 都走品牌色板**，不跟壁纸跑。
